# Contributing to MonoClaw

Thanks for your interest in improving MonoClaw.

## How to Contribute

1. **Fork** the repo
2. **Create a branch** for your change (`git checkout -b my-change`)
3. **Make your changes** — keep them focused and atomic
4. **Test** that the agent still works correctly with your changes (open in Claude Code, run a session)
5. **Submit a PR** with a clear description of what and why

## What We're Looking For

- New skills (`.claude/skills/your-skill/SKILL.md`)
- New job templates (`jobs/your-job/`)
- Improvements to memory system design
- Documentation fixes and clarifications
- Support for more AI agents beyond Claude Code

## Guidelines

- Keep files in markdown — no databases, no external services
- English only for all user-facing content
- Preserve the existing directory structure conventions
- Skills should be self-contained in their own directory
- Test your changes by running a real session with an AI agent

## Questions?

Open an issue. We'll get back to you.
